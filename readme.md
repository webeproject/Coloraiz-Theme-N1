#Coloraiz Theme | N1
A Coloraiz.css theme for <a href="https://nylas.com/">Nylas N1</a>
![Screenshot_1](Screen-azure-3.png)

##Change the color
Go to ```../styles/variables.less``` and at line 70 you can modify the var ```@azure-3``` to one of this:
**
 // @purple
- @purple-1
- @purple-2
- @purple-3
 
 // @lred
- @lred-1 
- @lred-2 
- @lred-3 
 
 // @marine
- @marine-1
- @marine-2
- @marine-3
 
 // @gray
- @gray-1 
- @gray-2 
- @gray-3 
 
 // @green
- @green-1 
- @green-2 
- @green-3 
 
 // @azure
- @azure-1 
- @azure-2 
- @azure-3 
 
 // @lgray
- @lgray-1
- @lgray-2
- @lgray-3
 
 // @blue
- @blue-1 
- @blue-2
- @blue-3
 
 //orange
- @orange-1
- @orange-2
- @orange-3
**
